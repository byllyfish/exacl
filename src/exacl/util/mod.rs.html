1//! Provides a cross-platform, minimal ACL API.
2//!
3//! Types:
4//!    `acl_t`
5//!    `acl_entry_t`
6//!
7//! Functions:
8//!    `xacl_init`      - create a new empty ACL
9//!    `xacl_free`      - destroy ACL
10//!    `xacl_foreach`   - apply a function to each entry in an ACL
11//!    `xacl_is_empty`  - return true if an ACL is empty
12//!    `xacl_is_posix`  - return true if ACL has Posix.1e semantics.
13//!    `xacl_add_entry` - append new entry to an ACL
14//!    `xacl_get_entry` - retrieve contents from an ACL entry
15//!    `xacl_get_file`  - get ACL from file path
16//!    `xacl_set_file`  - set ACL for file path
17//!    `xacl_is_nfs4`   - return true if file path uses `NFSv4` ACL on `FreeBSD`
18
